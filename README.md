Source of the cityofzion.io website

## Prerequisites

* Node.js 7 or 8
* yarn

## Getting started

    git clone https://github.com/CityOfZion/website.git
    cd website

    # Install the dependencies
    yarn

    # Run in dev mode (build and watch, may not work on Mac)
    yarn dev

    # Build and serve
    yarn serve

    # Run the tests
    yarn test

    # Just create a build
    yarn build
